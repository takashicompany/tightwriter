### オプション
|部品|個数|備考|
|:--|:--|:--|
|コンスルー|2|12pin、13pinに対応しておりますのでMCU(Pro Micro)にあわせて選択が可能です。|
|MXソケット|35|中央のキー以外は、キースイッチのはんだ付け/MXソケットによる挿し込みの両方に対応。|
|LED(WS2812B)|10|アンダーグロウライトとしてキーボードの底面を光らせることができます。|
